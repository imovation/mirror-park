# 智慧校园数据面板重构设计方案 (Panel Redesign Spec)

## 目标
根据最新的设计稿截图，全面重构 6 个专题的数据面板。确保覆盖截图中的**所有数据指标**，并在 UI 设计上“择优优化”——保留现有的“毛玻璃+双主题”质感，但全面吸收设计稿的排版、组件形态和信息密度。

## 核心架构约束
1. **全局布局**：维持 `ScreenLayout` 现有的左右两栏架构（SidePanel），不新增独立的大尺寸顶部贯穿数据栏。
2. **高度控制**：所有面板高度极度压缩，利用组件内轮播或内部滚动，严格**锁定在一屏内**，禁止出现全局垂直滚动条。
3. **视觉质感**：继续使用 `backdrop-filter` 模糊面板背景、圆角及柔和微阴影，不采用硬边框赛博风，以保证浅色模式的完美兼容。

## 全局通用组件重构 (UI Foundation)
为满足极高的空间利用率和视觉精致度，将新增或重构以下高复用 UI 组件：
- `<CompactStatsRow>`: 超紧凑横向数字流组件。用于收纳原设计稿中的“顶部核心指标”（Top Data Bar），并将其置于各专题侧边栏的顶端。
- `<RankList>`: 带彩色序号徽章（Badge）和行内数据进度条（Inline Bar）的排行列表。
- `<IconGrid>`: 带有发光态图标和紧凑数据的网格组件（用于“资产概况”等数据列举）。

## 各专题面板变动明细

### 1. 综合态势 (Overview)
- **左侧**：
  1. 【新增】顶部核心概况（占地面积、建筑面积、班级数、教职工、学生数）
  2. 人员构成（环形图+图例）
  3. 师资分布（条形图）
  4. 【调整】学生基础信息（由高占位桑基图降级为紧凑分布图）
- **右侧**：
  1. 活跃度时段统计（柱状/折线组合）
  2. 【新增】资产概况（IconGrid：电脑、空调、摄像头等）
  3. 【新增】功能室分布（条形排行图）
- **移除**：`BuildingDetail` 和目前的 `RecentActivity` 为新指标腾出空间。

### 2. 教学研究 (Teaching-Research)
- **调整**：将设计稿顶部 5 项指标（资源总量、课例成果、在研课题、名师工作室、近期更新）拆分重组到左右两侧最上方。
- **升级**：全面升级“名师工作室”和“资源更新动态”的列表UI，使用新版 `RankList` 展现多列对齐排版。

### 3. 智慧图书 (Library)
- **左侧**：概况指标、借阅统计（图表）、热门图书 TOP10 排行榜（`<RankList>`）、班级借阅排行。
- **右侧**：部类借阅占比（环形饼图）、入馆统计（折线图）。
- **新增**：“新书/好书推荐”，替换现有表现形式，强化推荐位视觉。

### 4. 行政办公 (Admin) & 智慧教学 (Academics)
- **图表极简压缩**：废弃占用大面积的传统仪表盘（Gauge），使用带有中心数据或半环形态的极简进度条图表（Mini-progress/Ring）来展示“教室使用率”等指标。
- **指标前置**：各年级课程排课数、总数等核心指标优先使用 `<CompactStatsRow>`。

### 5. 智慧安防 (Security)
- **列表精绘**：安防截图中有大量分类比对（门禁通行分段、请假类型）。引入带数据条的富文本列表，代替纯 Echarts 柱状图，提高单面板指标承载量。

## 边界情况与防御
- 所有新增的 `RankList` 和文本域必须具有 `text-overflow: ellipsis` 以防止长名称撑破高度。
- 当 API 数据为空时，所有新图表组件必须平滑降级展示 `StatusPanel` 的 `empty` 态。
