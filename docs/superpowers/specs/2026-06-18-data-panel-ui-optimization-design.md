# 智慧校园可视化平台 - 数据面板 UI 优化与数据覆盖设计文档

## 1. 目标与范围
基于产品设计团队提供的 6 张最终效果图（位于 `docs/panmd/`），对平台的数据面板进行彻底的 UI 优化，并确保 6 个专题（综合态势、教学研究、行政办公、智慧图书、智慧教学、智慧安防）的所有数据指标 100% 覆盖。

## 2. 总体设计原则
*   **统一的视觉语言**：深色科技风（Cyberpunk / 科技蓝）。
*   **质感升级**：引入毛玻璃滤镜（backdrop-filter）、半透明渐变背景、科技感发光边框。
*   **图表重构**：全面升级 ECharts 主题，使用渐变色、发光线条（neon glow）、阴影高光等表现手法。
*   **模块化组件**：先提取公共组件（面板容器、滚动列表、各类高频图表），再进行页面拼装。

## 3. 核心基础组件重构 (UI Foundation)
### 3.1 科技感面板容器 (`DashboardPanel`)
*   **背景**：带有透明度的深蓝渐变 (`linear-gradient`)，配合 `backdrop-filter: blur(10px)`。
*   **边框修饰**：利用 CSS 伪元素绘制四个边角的科技感折角修饰线（青蓝色）。
*   **标题栏**：左侧增设青色发光修饰块，标题文字采用无衬线字体，并带有微弱的 `text-shadow` 科技光效。

### 3.2 ECharts 图表主题升级 (`Neon Chart Theme`)
*   **配色方案**：青蓝（主色）、霓虹紫、亮黄、翠绿。
*   **柱状图 (Bar)**：渐变填充，顶部带高光或切面效果。
*   **折线图 (Line)**：启用 `shadowColor` 和 `shadowBlur` 制造线条发光，下方区域采用半透明渐变填充。
*   **环形图 (Pie/Ring)**：细化边框线，内部增加科技感装饰刻度环。

### 3.3 高频通用组件
*   **Top 核心指标卡**：发光图标容器 + 科技字体数字（结合现有 NumberFlip） + 数据辅助标签。
*   **科技感滚动列表 (`ScrollList`)**：新增深色半透明表头，列表行带悬浮高亮光效，排行前三名带有金银铜专属配色标识。
*   **垂直时间轴 (`Timeline`)**：用于校历日程等功能，带发光时间节点和渐变连接线。

## 4. 6 大专题数据指标全覆盖矩阵

### 4.1 综合态势 (Overview)
*   **Top 核心指标**：占地面积、建筑面积、班级数、教职工、学生数。
*   **左侧**：人员构成（环形图+条形图）、师资分布（横向渐变进度条）、学生基础信息（男女对比柱状图）。
*   **右侧**：活跃度时段统计（柱状图）、资产概况（六宫格卡片）、功能室分布（条形图）。

### 4.2 教学研究 (Teaching Research)
*   **Top 核心指标**：资源总量、课例成果、在研课题、名师工作室、近期更新。
*   **左侧**：教学资源（6项分类卡片）、资源统计（三大块展示）、资源更新动态（自适应滚动列表）。
*   **右侧**：教师课题统计卡、课题项目详情（图文排版）、名师工作室（高亮排版列表）。

### 4.3 行政办公 (Administrative Office)
*   **Top 核心指标**：部门数量、教职工总数、今日出勤率、功能室数量。
*   **左侧**：通知公告（带Tag的滚动列表）、值班安排（联系方式卡片组）、会议管理（概览+状态列表）。
*   **右侧**：校历日程（垂直时间轴）、教职工考勤（综合指标+1-6月考勤折线图）。

### 4.4 智慧图书 (Smart Library)
*   **Top 核心指标**：纸质图书、电子图书、期刊种类、报纸种类。
*   **左侧**：借阅统计（概览+7日借阅柱状图）、热门图书 TOP10（排行条形图）、班级借阅排行（排行条形图）。
*   **右侧**：图书部类借阅占比（多色霓虹环形图）、入馆统计（人次曲线面积图）、新书/好书推荐（图文列表）。

### 4.5 智慧教学 (Smart Teaching)
*   **Top 核心指标**：今日课程、正在上课、教室总数、教室使用率、班级总数。
*   **左侧**：课程安排（课程数、学科环形图、排列表）、教室使用（双环形图+楼栋使用率条形图）、班级管理（表格列表）。
*   **右侧**：学生出勤（大环形图+年级柱状图+排行条形图+趋势折线图）、考试管理（平均分组合图表）、教学设备（状态环形图）。

### 4.6 智慧安防 (Smart Security)
*   **Top 核心指标**：监控设备、门禁设备、今日告警、今日访客。
*   **左侧**：监控状态（覆盖率条形图+仪表盘）、门禁管理（折柱混合图+异常记录列表）、学生请假管理（环形图+离校记录列表）。
*   **右侧**：访客管理（占比环形图+登记列表）、告警事件（双环形图+告警滚动列表）、食堂安全（就餐人数柱状图+实时监控视频占位窗+检查记录）。

## 5. 实施策略
遵循“基础组件先行，后全局替换”的策略：先完成 `DashboardPanel` 和 `ECharts` 等核心UI层的重构，然后逐个覆盖6大专题的数据面板页面。
